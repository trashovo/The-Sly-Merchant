# The-Sly-Merchant

Программа для моделирования циклического удаления элементов на базе цикличного списка. Написана в рамках сессионного задания на 1 курсе.

### Архитектура проекта

Проект спроектирован с разделением логики по модулям (файлам):

*  **main.py** — точка входа, инициализирует приложение и запускает главный цикл программы.
*  **interface.py**  — отвечает за интерактивный текстовый интерфейс меню и отображение текущих переменных.
*  **list.py** — содержит реализацию цикличного списка и сам алгоритм циклического исключения элементов.
*  **user_input.py** — отвечает за прием данных от пользователя и защиту от некорректного ввода.

### Как запустить проект

Запустить main.py
